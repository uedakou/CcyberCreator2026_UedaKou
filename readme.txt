◆このゲームについて
・・・
C++ 2作品目
バトルモービルをリメイクした
レース風アクションゲームです。

◆起動方法
・・・
exeファイルを起動

◆基本操作
・・・
ゲーム
	移動	: WASD
	攻撃	: space
タイトル
	移動	: WS
	決定	: enter

◆実装内容
・・・
ストラテジーパターンによる管理シーン
オブジェクトのリスト構造
キャラクターを状態パターンで制御
キャラクターのモーション
オブジェクトのソート
